Building production AI systems for healthcare. Currently maintaining two systems: a clinical platform (10 modules, active pilot) and open-source LLM trust infrastructure (hybrid RAG, model routing, citation verification, multi-tenant API).

## Open Source

- [llm-trust-layer](https://github.com/nomad-link-id/llm-trust-layer) — End-to-end LLM trust pipeline: query routing + hybrid retrieval + citation verification in one call
- [hybrid-rag-pipeline](https://github.com/nomad-link-id/hybrid-rag-pipeline) — BM25 + semantic search + Reciprocal Rank Fusion with authority boosting and recency weighting
- [citation-guard](https://github.com/nomad-link-id/citation-guard) — Post-generation citation verification for RAG systems. Deterministic, <5ms overhead
- [model-router-chain](https://github.com/nomad-link-id/model-router-chain) — Complexity-based LLM routing with automatic fallback chain
